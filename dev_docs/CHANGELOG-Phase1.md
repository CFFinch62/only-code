# Changelog - Phase 1: Core Editor

## [0.2.0] - 2025-12-04

### Added
- `OnlyCodeEditor` widget with syntax highlighting (default Python).
- `MainScreen` layout with Header, Editor, StatusBar, and Footer.
- `StatusBar` showing file path, cursor position, and modification status.
- File operations: New, Open, Save, Save As.
- File dialogs using `DirectoryTree`.
- Keybindings: `Ctrl+O` (Open), `Ctrl+S` (Save), `Ctrl+N` (New), `Ctrl+Q` (Quit).
